== Description ==

April 23rd 2014 - Coull no longer supports the Vidlinkr plugin and it has now been disabled. If you have previously installed this plugin, the embedded videos on your website will be unaffected by this update and you don't need to do anything.

We've disabled the plugin because we've changed the way Coull Vidlinkr works. If you're a WordPress publisher looking to monetize your video content you can <a href="http://coull.com/publishers">find out more about Coull Vidlinkr here</a>.
